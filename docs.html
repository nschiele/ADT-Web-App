Introduction

ADTapp is a tool that enables you to design and visualize attack-defense scenarios using the Attack-Defense Tree model. The tool provides an intuitive interface that allows you to easily create and modify Attack-defense Trees, and generate a custom tree using xml, json and plain text. With ADTapp, you can quickly generate visual representations of your attack-defense scenarios and identify potential vulnerabilities and mitigations.
